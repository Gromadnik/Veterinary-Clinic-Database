# Veterinary-Clinic-Database

PROJECT OVERVIEW
Design and implementation of a relational database system for a veterinary clinic. The project covers the full lifecycle from business requirements and ERD modeling to physical implementation via DDL scripts.

KEY FEATURES
- Management of patient records (animals), owners, and medical staff.
- Tracking of visits, diagnoses, and prescribed treatments.
- Structured billing and pharmaceutical inventory management.
- Implementation of business rules through relational constraints.

TECHNICAL SPECIFICATIONS
- Modeling: Conceptual (ERD) and Logical relational models.
- Normalization: Database designed according to 3rd Normal Form (3NF).
- Implementation: SQL DDL scripts for table creation and relationship definitions.
